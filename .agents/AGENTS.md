# Design Guidelines

1. **Firm Design (Glassmorphism/Neumorphism)**: Always use the corporate design style when adding new UI elements. Never use native browser inputs like `<input type="time" />`, `<input type="date" />`, or default browser selects if it breaks the visual harmony. Rely on existing custom UI components (e.g., `CustomSelect`, `CustomDatePicker`, `GlassCard`, `neu-input`, `pill-btn`) or build new custom components that strictly follow the project's glassmorphism and neumorphism CSS style guide.
